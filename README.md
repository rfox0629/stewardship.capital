# Stewardship.Capital

One-page website for Stewardship.Capital, the acquisition and transition arm helping business owners steward what they have built for lasting Kingdom impact.

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
